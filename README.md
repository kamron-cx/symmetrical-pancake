# Instagram Downloader Bot

Ushbu bot Instagram-dan video va rasmlarni yuklab olish uchun mo'ljallangan.

## O'rnatish:
1. Loyihani yuklab oling.
2. `pip install -r requirements.txt` buyrug'ini bajaring.
3. `bot.py` faylidagi token o'rniga o'z tokeningizni qo'ying.
4. Botni ishga tushiring: `python bot.py`
